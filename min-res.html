Four Column Grid

Minimal Responsive's layout configurations are limited to four columns for the sake of simplicity. At less than 568px all columns are full width and stacked.

Sidebar style layouts are acheived by using the .two-three-column and .three-four-column classes, bearing in mind the document flow and it's implications on these layouts on small screens.

The .container class is used to center the columns on the page so all column's should be nested inside of a .container.

Rows should be filled completely to ensure proper stacking. Empty column's can be used as placeholders.

Nesting Columns

Columns can be nested by placing a <div> with the .nest class around columns that you want nested. This allows for more robust layouts in an otherwise simple column framework.

When nesting columns as when creating sidebar style layouts, it is important to keep the document flow in mind. Nesting is supported up to two levels.

Full Width Images

Modern web design relies heavily on imagery to make an impact. As a result, a high priority was placed on including responsive, full width images in Minimal Responsive.

For full width images, simply give a <div> element the .fw-img class and a #unique-id. The .image class takes care of positioning and sizing the image.

In your CSS, set the id's background-image and give it a height. Defining a height is very important, otherwise the image will not be displayed.

{HTML}
<section> <div class="fw-img" id="some-id"> </div> </section>
{CSS}
#some-id{ background-image:url('img/forest.jpg'); height:35em; }

Column Width Images

Minimal Responsive also has a built in class to handle column width images. Just place an <img> element inside of a column, give it the .cw-img class, and the image will fill it's parent column.

{HTML}
<div class="one-column"> <img class="cw-img" src="img/cold-mountains.jpg"> </div>

Vertical Centering

Minimal responsive has built in vertical centering classes. To center content, first create one <div> with the class .vcenter-wrap and then another with the class .vcenter-box inside of it.

In order to give .vcenter-wrap a background-image, add the .image class and give it a #unique-id. Then give the id a background-image and height in your CSS. It is important to set a height.

Columns can be placed inside of the .vcenter-box, allowing multi-column layouts to be vertically centered in an area on the page. However, use caution as the .vcenter-wrap page area has a fixed height and depending on browser width, the content may overflow.

{HTML}
<div class="vcenter-wrap image" id="some-id"> <div class="vcenter-box"> <h3>Vertically Centered<h3> </div> </div>
{CSS}
#some-id { background-image:url('img/ocean.jpg'); height:35em; }

Buttons

Minimal Reponsive has consistently styled buttons for <a>nchors, <button>, <input type=submit>, and <input type=button> elements.

The <button>, <input type=submit>, and <input type=button> elements are already styled without the need to add a class.

<a>nchors need the .button class. For a bright button, add the .bright class to any of the button types.

{HTML}

Tables

Tables are minimally styled and optimized for horizontal interpretation. Emphasis was placed on appropiately padding table cells for easy readability.

When constructing tables, it is necessary to use <thead> and <tbody> tags to benefit from Minimal Responsive's pre-built styles.

Forms

Several form <input>'s are style to match Minimal Responsive's look and feel. Form <input>'s can be nested inside of columns in order to achieve the desired form layout.

Keep in mind that all columns are stacked on mobile devices. Form fields should follow left-to-right document flow in order to ensure proper stacking.
